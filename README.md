## Core idea

Constraints reduce a space without forcing collapse.

### In geometry
- random vectors → orthogonal  
- fixed thresholds → exponential selectivity  
- signal survives if scaled ~ 1/√d  

### In arithmetic
- bounded modular filters → exact density  
- cumulative filters → multiplicative decay
