Summary

This project studies constraint-first reconstruction of geometry using multiple probe observables (EE, WL, GEO).

Earlier results show that matching observable values alone can leave ambiguity. Adding a derivative constraint enforces local structure, and a probe sweep reveals that not all observables are required.

In this setup, the minimal identifying set is:

EE ⊕ GEO ⊕ derivative

Context

This work was developed alongside experiments in neural reconstruction of geometry under shared multi-slice constraints.

It connects to questions raised in recent arXiv work on multi-observable reconstruction:

arXiv:2604.05970
